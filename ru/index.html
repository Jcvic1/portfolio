Описание

Веб-приложение для потоковой передачи музыки из яндекс музыки с приложением-чатом, похожим на среду, соединяющую нескольких пользователей яндекс музыки для обмена музыкой, прослушивания треков друг друга. Это строго основанное на django приложение без front-end фреймворка, что приводит к сложному использованию шаблонов django для достижения желаемого внешнего вида. Он работает на платформе django framework, используя готовую среду для запуска приложения, он использует внешнюю базу данных на postgresql для хранения данных и aws bucket s3 для хранения мультимедиа, таких как фотографии профиля.
